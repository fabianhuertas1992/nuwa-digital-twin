# Resumen de cambios - 2026-01-14

## Panorama general
- Se incorporaron scripts para análisis local de fincas (NDVI, deforestación EUDR y línea base de carbono) con soporte para múltiples formatos GeoJSON.
- Se añadió un flujo de procesamiento por lotes para analizar múltiples fincas y generar resúmenes en CSV/JSON.
- Se integraron utilidades para subir análisis a IPFS (Pinata) y verificar accesibilidad de hashes en múltiples gateways.
- Se agregaron datos de entrada/salida de ejemplo y resultados de análisis listos para revisión.
- Se actualizaron dependencias de Python para soportar las nuevas utilidades (requests, python-dotenv).

## Cambios principales
### Scripts de análisis local
- `scripts/local-analysis/analyze_farm.py`: analiza fincas desde archivos locales con soporte para Feature/FeatureCollection/MultiPolygon y salida normalizada.
- `scripts/local-analysis/batch_process.py`: procesamiento en lote con generación de resultados por finca y resúmenes agregados.

### IPFS
- `scripts/ipfs/upload_to_ipfs.py`: carga de análisis a IPFS vía Pinata y generación de metadata NFT (CIP-25).
- `scripts/ipfs/verify_ipfs.py`: verificación de accesibilidad de hashes IPFS en múltiples gateways.

### Datos y salidas de ejemplo
- Nuevos GeoJSON/JSON de fincas en `data/farms/input/`.
- Resultados de análisis y manifest IPFS en `data/farms/output/`.

### Dependencias
- `python-scripts/requirements.txt`: se añadieron `requests` y `python-dotenv` para soporte de IPFS y configuración.
